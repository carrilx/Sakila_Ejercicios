# Base de Datos Sakila

Esta carpeta contiene los archivos SQL de la base de datos Sakila.

## Archivos

- `sakila-schema.sql` — Estructura de la base de datos (tablas, índices, vistas, procedimientos)
- `sakila-data.sql` — Datos de ejemplo

## Instalación

```bash
mysql -u root -p < sakila-schema.sql
mysql -u root -p < sakila-data.sql
```

## Descarga oficial

Si no tienes los archivos, puedes descargarlos desde la web oficial de MySQL:
https://dev.mysql.com/doc/index-other.html
